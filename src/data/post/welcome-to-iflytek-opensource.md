# 欢迎来到 iFLYTEK 开源社区

我们很高兴地宣布，iFLYTEK 开源官网正式上线！

## 我们的使命

科大讯飞一直致力于推动人工智能技术的发展和应用。通过开源，我们希望：

- **共享技术成果**：将我们在 AI、NLP、语音识别等领域的技术积累以开源形式分享给社区
- **促进协作创新**：与全球开发者共同推动技术进步
- **培养开源文化**：建立健康、可持续的开源生态

## 开源项目

目前我们正在逐步开放以下领域的开源项目：

- **AI 平台**：大模型相关的开源工具和框架
- **自然语言处理**：NLP 引擎和工具集
- **语音技术**：语音识别与合成 SDK
- **开发者工具**：各类 SDK 和开发辅助工具

## 参与贡献

我们欢迎各种形式的贡献：

1. **代码贡献**：提交 Bug 修复、新功能开发
2. **文档贡献**：完善文档、编写教程
3. **社区建设**：参与讨论、组织活动
4. **反馈建议**：提出功能建议、使用反馈

请访问 [贡献指南](/contribute) 了解更多信息。

## 未来计划

我们将持续更新网站内容，包括：

- 更多开源项目展示
- 技术博客和教程
- 社区活动信息
- 开源全景图（Landscape）

感谢你对 iFLYTEK 开源社区的关注！
